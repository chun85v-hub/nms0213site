1

직접생산확인증명서 자가 진단

철제 안내판을 등록하기 위해서는 '금속 가공' 관련 설비가 필수입니다. 현재 (주)엔엠에스의 상태를 확인합니다.

✓ 확인 완료

  • 보유 증명서: 직접생산확인증명서 (안내판)
  • 필수 설비 확인: 용접기 (전기/아르곤), 금속 절단기 보유 확인됨
  • 조치 사항: 별도의 시설 변경 신청 없이 즉시 품목 추가 가능
2

규격서 및 도면 작성

조달청 제출용 규격서는 일반 시방서보다 구체적이어야 합니다. 아래 내용을 포함하세요.

① 재질 표기: 단순 '철재'가 아닌 일반구조용 압연강재(SS400) 또는 아연도금강판(EGI) 등 KS 규격 명시

② 마감 처리: 분체도장 (두께 60μm 이상), UV 인쇄 후 투명 코팅 등 내구성 관련 내용 필수

③ 도면 준비: 정면도, 측면도, 평면도 + 앙카 상세도 (CAD -> PDF 변환)

3

나라장터 시스템 입력

서류 준비 후 시스템 접속 절차입니다.

1. 나라장터 로그인 > [계약관리]

2. [다수공급자계약] > [계약변경신청]

3. [규격추가] 탭 선택

4. 신규 모델 정보 입력 및 규격서/도면/실적증명 파일 업로드
